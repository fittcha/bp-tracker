# 2026 ZEST BP Tracker 사용 가이드

## 앱 설치 방법

### iPhone (Safari)
1. Safari에서 https://bp-tracker-six-kohl.vercel.app 접속
2. 하단 공유 버튼 (네모+화살표) 탭
3. "홈 화면에 추가" 선택
4. "추가" 탭 → 홈 화면에 앱 아이콘 생성

### Android (Chrome)
1. Chrome에서 https://bp-tracker-six-kohl.vercel.app 접속
2. 우측 상단 메뉴 (점 3개) 탭
3. "홈 화면에 추가" 또는 "앱 설치" 선택
4. "설치" 탭 → 홈 화면에 앱 아이콘 생성

> 설치 후 일반 앱처럼 전체 화면으로 사용할 수 있습니다.

---

## 로그인

- 본인 username 입력 → "로그인" 탭
- 첫 로그인 시 4자리 PIN 설정 (다음부터 이 PIN으로 로그인)
- "자동 로그인" 체크하면 다음부터 자동 진입

## 화면 설명

### 홈 (대시보드)
- **D-day**: 촬영일(6/20)까지 남은 일수
- **프로그램 진행**: 5단계 타임라인 (Reset Block → Make Up)
- **오늘의 기록**: 운동/일일기록 진행 상태
- **이번 주 요약**: 체중 변화, 평균 칼로리, 수면, 운동 완료율

### 운동
- 상단에서 날짜/요일 선택
- 코치 운동 목록이 자동으로 표시됨
- 각 운동 왼쪽 체크로 완료 표시
- 오른쪽 "lb" 버튼으로 무게 입력 (+/- 5lb 단위)
- 하단에서 개인 운동 추가 가능
- 모든 변경은 자동 저장

### 기록 (일일 기록)
- **체중**: kg 단위 입력
- **수면**: 취침/기상 시간 선택 → 총 수면시간 자동 계산
- **운동 여부**: O/X 선택
- **당/가공식품**: 미섭취(X) 또는 섭취 항목 입력
- **식단**: 식단 앱 캡처 이미지 업로드 → 칼로리/매크로 자동 추출 (수동 수정 가능)
- **영양제**: 6종 체크리스트
- **수분**: 물컵 클릭 (250ml 단위, 최대 2L)
- **메모**: 자유 메모
- 모든 변경은 자동 저장
- 하단 카톡 공유 텍스트 자동 생성 → 복사 버튼

### 요약
- 주차 선택 → 해당 주간 데이터 확인
- 체중 변화 차트
- 평균 매크로 비율 차트
- 주간 통계 (칼로리, 수면, 운동, 당 섭취)
- 하단 로그아웃 버튼

## 카톡 공유

기록 탭에서 데이터 입력 후 하단에 자동 생성되는 텍스트를 "복사" 버튼으로 복사하여 카톡에 붙여넣기하면 됩니다.

예시:
```
3/10(화)
총 수면 시간 : 7시간 15분 (23:00 / 06:15)
운동 여부 : O
당/가공식품 섭취 여부 : X
```
