## 6. 컬러 테마 — 골드 전환

### 흐름
- 탭 활성색: 주황(#F97316) → 블루(#2B6CB0) 시도 → **주황 환원**(사용자: "주황이 낫다") → 골드 로고에 맞춰 **앰버골드 전면 전환**.

- 골드는 **어두운 배경(로고)에서 밝게(#C9A227), 흰 배경(인앱)에서 다운톤(#C0974A)** — 같은 색조를 배경별로 튜닝.

### 골드 체크박스 실험 → 부분 환원
- `--success`(완료/활성 공통색)를 골드(#C0974A)로 통일 시도 → "별로 안이쁘다".
- **운동 그룹 체크박스 등 모든 체크박스/완료표시는 파랑(#3B82F6)으로 환원**.
- **계산기 "직접입력/1RM" + 결과만 골드 유지** — `Calculator.tsx`를 `success` → `accent-pop`으로 분리.
